# OIAX Docker Image for Re:VIEW (digital publishing system)

## Summary

* Debian (Stretch) slim
* Ruby 2.3.3
* Node.js 10
* Re:View 2.0.0 (patched by https://github.com/kuroda/review)

## Building a docker image

```
$ docker build -t oiax/review-customized .
```
